# sip_ua Package Notes
- Version: 1.1.0
- SIP over WebSocket && TCP
- Audio/video calls via flutter-webrtc
- Supports Asterisk
- WebRTC requires DTLS/SRTP
- Codecs: opus, G722, ILBC, PCMU, PCMA, etc.
- IMPORTANT: Asterisk must be configured with DTLS/SRTP for WebRTC
- Need to configure certificates for DTLS
